# Game of War

A simple implementation of the classic card game War.

## Screenshots

![screenshot](/preview.png)

## Features

- Play against the computer
- Supports consecutive wars
- LocalStorage persistence
- Responsive layout

## Built With

- HTML
- CSS
- JavaScript
- Deck of Cards API

## Credits

Card resources provided by Scrimba's API, based on Deck of Cards API.
